# Basic-Expense-Tracker

Basic Expense Tracker

Project Overview

Basic Expense Tracker is a basic desktop application developed using python, Tkinter and SQLite database. The application is for recording income and expenses, classifying transactions and monitoring finances via a simple graphical interface.

The project was created as an assignment of the PFD200 Professional Development Module to enhance proficiency in software development and database integration testing.

Features
-> Add income and expense transactions
-> Categorise transactions
-> Store data in an SQLite database
-> View transaction history
-> Delete transactions
-> Calculate total income
-> Calculate total expenses
-> Display current balance
-> User-friendly graphical interface
